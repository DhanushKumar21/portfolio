Adv. Diploma — Final Project 2024

Horror FPS — Survive the Dark

A first-person psychological horror game built in Unreal Engine 5 featuring dynamic AI enemies, atmospheric lighting, and environmental storytelling.

The game challenges the player to navigate a haunted manor while evading AI-driven enemies that use Behaviour Trees and Perception components to track sound and sight. I designed all the Blueprint logic for player mechanics (crouching, sprint, heartbeat system), enemy patrol and chase states, interactable objects, and the main menu flow. Lumen was used for dynamic global illumination to achieve the moody atmosphere without baking. The project also features a custom inventory/item system built in C++ exposed to Blueprint.

Unity 2D2024

2D Platformer — Action Adventure

A Hollow Knight-inspired 2D action platformer built in Unity featuring a full combat system, enemy AI, boss fights, and a drag-and-drop inventory.

Built entirely in C# from scratch. Systems include: variable-jump mechanics, three-hit melee combo with pogo bounce, dash / ground-slam / cyclone slash abilities with an energy resource system, loot drops, a slot-based drag-and-drop inventory using ScriptableObjects, an NPC shop with dialogue, multiple trap types, four enemy archetypes, and a two-phase boss with randomised attacks and a stagger system.
